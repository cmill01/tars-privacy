# TARS Privacy Policy

**Last Updated:** February 16, 2026

## Overview

TARS is a personal AI assistant built using OpenClaw. This privacy policy describes how data is handled.

## What We Collect

- **Phone Numbers:** Only for SMS and voice call functionality
- **Message Content:** Call and SMS logs stored locally on secure server
- **Timestamps:** For message/call history

## What We Don't Do

- ❌ **No data selling** - Your data is never sold to third parties
- ❌ **No third-party sharing** - Data stays on private server
- ❌ **No marketing** - This is a personal assistant, not a marketing tool
- ❌ **No tracking** - No analytics or third-party trackers

## Data Storage

- All data stored on private VPS
- Credentials stored locally in encrypted config files
- No cloud storage of personal information

## Data Retention

- Call/SMS logs retained for operational purposes
- You can request deletion at any time

## Third-Party Services

TARS uses these services for functionality:
- **Twilio:** Phone/SMS infrastructure (see [Twilio Privacy](https://www.twilio.com/legal/privacy))
- **ElevenLabs:** Text-to-speech (see [ElevenLabs Privacy](https://elevenlabs.io/privacy))

## Your Rights

You have the right to:
- Request a copy of your data
- Request deletion of your data
- Opt out at any time

## Contact

For privacy questions or data requests, contact the system administrator.

## Use Case

TARS is a **personal use AI assistant** for:
- Voice calls
- SMS notifications
- Personal reminders
- Home automation
- General assistance

**Not used for:** Marketing, sales, spam, or commercial messaging.

## Compliance

- A2P 10DLC Registration: Low Volume Mixed Use
- Message volume: <10 messages/day
- Personal/individual use only

*This is a personal project. All messaging is opt-in and for individual use only.*
